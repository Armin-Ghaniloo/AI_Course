# Project 04 – Classical Machine Learning

This project was completed as part of the Artificial Intelligence course during my Computer Engineering studies at the University of Tehran.

The objective of this project is to implement, evaluate, and compare several classical machine learning algorithms while gaining practical experience with data preprocessing, feature engineering, and model evaluation.

---

## Project Overview

The project consists of four independent machine learning tasks covering different classification problems and implementation techniques.

### Part 1 – Spam Message Detection

Implemented and compared multiple models for spam message classification.

Algorithms:

- Decision Tree (implemented from scratch)
- Gaussian Naive Bayes
- Random Forest
- GridSearchCV for hyperparameter tuning

Concepts:

- Entropy
- Information Gain
- Decision Trees
- Model Evaluation
- Accuracy, Precision, Recall, F1-score

---

### Part 2 – Softmax Regression from Scratch

Implemented a multiclass Softmax Regression model using NumPy without relying on machine learning libraries.

Concepts:

- Gradient Descent
- Softmax Function
- Cross-Entropy Loss
- One-Hot Encoding
- Multiclass Classification

The custom implementation is also compared with Scikit-learn's Logistic Regression.

---

### Part 3 – Mountain Route Safety Prediction

Built a Decision Tree classifier to predict safe hiking routes.

Concepts:

- Data Cleaning
- Feature Engineering
- One-Hot Encoding
- Standardization
- Decision Trees
- Performance Evaluation

---

### Part 4 – Music Genre Classification

Implemented a K-Nearest Neighbors (KNN) classifier for music genre prediction.

Concepts:

- KNN Classification
- Distance-Based Learning
- Feature Scaling
- Bias-Variance Trade-off
- Model Comparison
