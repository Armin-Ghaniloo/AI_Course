# Blood Cell Classification with PyTorch

This project was developed as part of my Artificial Intelligence course at the University of Tehran.

The project introduces the fundamentals of Deep Learning using PyTorch and compares the performance of a Multi-Layer Perceptron (MLP) and a Convolutional Neural Network (CNN) for blood cell image classification.

---

## Topics Covered

- PyTorch fundamentals
- Tensor operations
- Custom Dataset and DataLoader
- Neural Network implementation
- Multi-Layer Perceptron (MLP)
- Convolutional Neural Network (CNN)
- Model training and evaluation
- Confusion Matrix
- Classification Report
- Learning Curves
- Misclassified sample analysis

---

## Models

- Multi-Layer Perceptron (MLP)
- Convolutional Neural Network (CNN)

## Results

- MLP Accuracy: **84%**
- CNN Accuracy: **98%**

The CNN significantly outperformed the MLP by learning spatial features directly from the input images.
