# Checkers AI using Minimax and Alpha-Beta Pruning

This project was developed as part of an Artificial Intelligence course at the University of Tehran.

The project implements an intelligent Checkers (Draughts) player using the Minimax search algorithm and an optimized Alpha-Beta Pruning version. A heuristic evaluation function is also designed to improve the AI's playing performance.

## Results

The Alpha-Beta Pruning implementation significantly reduces the search time compared to the standard Minimax algorithm while producing identical game decisions.

Increasing the search depth improves the AI's playing strength, and Alpha-Beta Pruning allows deeper searches within practical execution times.
